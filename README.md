# 🚀 React Multi-Page Router App

## 📖 Description
This project is a **Multi-Page Single Page Application (SPA)** built using React Router. It demonstrates seamless navigation between different pages without reloading the browser, improving performance and user experience.

---

## ✨ Features
- 🏠 Home, 👩‍💻 About, 📞 Contact pages
- 🔄 Client-side routing using React Router
- 🎨 Clean and modern user interface
- 🧾 Interactive contact form
- ❌ Custom 404 Not Found page

---

## 🛠 Technologies Used
- React JS
- React Router DOM
- JavaScript (ES6)
- CSS3

---

## 🚀 Getting Started

### Install dependencies

npm install
Run the application
npm start
